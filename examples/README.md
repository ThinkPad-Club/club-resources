# Coding Examples 📝

This folder contains example code from our daily coding demos. You’re welcome to open these files and keep them on your screen while we’re coding so you can see what the code should look like, catch up if you fall behind, fix bugs, or check your work. Getting stuck, falling behind, or breaking your code is totally normal when learning to program—it happens to everyone! These examples are here to help you keep going. Just remember not to copy and paste the code. Instead, read it, think about what it does, and type it yourself. Typing the code helps your brain learn how programming works. The goal isn’t to finish fast—the goal is to learn, experiment, and try your best.

1. Open the example for the demonstration
2. Keep it visible while coding
3. Look at the example if you get stuck
4. Don't copy / paste code
5. Start over if you're stuck too long
6. Have fun and get creative : )
